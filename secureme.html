SECUREME

INTERACTION DESIGN

SecureMe provides a security system for AirBnb hosts and guests. With SecureMe, hosts would feel more comfortable managing their properties, and guests would feel safer and in control in an unfamiliar environment. This is an ecosystem of different interfaces, allowing multiple types of interactions.

Collaborated with Johnny Knusten and Veslava Ovendale

AUDIO INTERFACE

The guests are greeted with the voice interface when they first enter the home, and it is also a voice assistant for guests. The audio system acts as the alarm system as well, when the sensors around the home detect unauthorized attempts to enter.

MAIN INTERFACE

The main interface is normally located in easily accessible place in the home, allowing guests use with ease. It features two main views: the floor plan of the home, and a map around the house.

LIGHTS & LOCKS

Guests can control lights and locks remotely with the main interface. This also allows guests to familiarize with the locations of the lights and locks in the home.

SECURE PLACES

Guests can store their valubales in these places in the home that hosts have marked as safes or hidden places.

MAP

Hosts and previous guests are welcomed to leave notes and locations on the map for guests to explore the area as well as avoid certain places in the neighborhood.

APP INTERFACE

SecureMe is directly integrated into the AirBnb app, and guests and hosts would both receive notifications from the system. Guests can also access the floor plan view from the main interface from the app during their stay.

APP

An added feature to the existing AirBnb app is a thumbprint scan or pin code for guests to access the home without a key.

ALERT SYSTEM

When there is an attempted intrusion, the main interface would lockdown, the audio system would deter the intruder away, and the AirBnb app would notify the guests.
